# Admin & Moderation Tools

RetroChat provides a comprehensive set of tools for administrators and moderators to keep the platform safe, organized, and running smoothly.

---

## Roles Overview

RetroChat has several user roles, each with different levels of access:

| Role | Description |
|------|-------------|
| **User** | Standard user — can chat, join rooms, and use platform features |
| **Community Leader** | Can manage specific rooms they've been approved to lead |
| **Moderator** | Can moderate content and users across assigned areas |
| **Admin** | Full platform access — can manage all rooms, users, settings, and configurations |

---

## Admin Dashboard

Administrators have access to a full-featured dashboard with the following tabs:

### Users Tab
- View all registered users
- Search users by screen name or email
- View user profiles and activity
- Manage user accounts (suspend, ban, etc.)

### Rooms Tab
- View all chat rooms on the platform
- See room statistics (message count, user count)
- Delete or modify rooms
- Mute rooms to prevent messaging

### Communities Tab
- View all communities
- Manage community settings and visibility
- Review community reports

### Retro TV Management
- **Videos Tab** — Manage the video library for Retro TV channels
- **Channel Groups Tab** — Organize channels into groups
- **IPTV Tab** — Configure IPTV stream sources

### Settings Tab
- Configure platform-wide settings
- Manage maintenance mode
- Schedule maintenance windows
- View and manage system health

### Actions Tab
- View admin action history
- Track all administrative actions taken on the platform

## Moderator Tools

Moderators have access to a subset of admin tools:

### Message Moderation
- Delete inappropriate messages in chat rooms
- Messages can be deleted by room creators, assigned moderators, or admins

### Room Moderation
- Mute rooms to temporarily prevent all messaging
- Ban users from specific rooms
- View banned user lists
- Manage room settings (description, tags, wallpaper)

### Community Moderation
- Review and act on reported posts
- Remove inappropriate community posts
- Manage community members
- Pin and unpin important posts

---

## Community Leader System

Community Leaders are trusted users who manage specific chat rooms.

### Requesting Community Leader Status
1. Navigate to a chat room you'd like to manage.
2. Click **Request Community Leader**.
3. Provide a reason for your request.
4. An admin will review and approve or deny the request.

### Community Leader Permissions
- Manage room settings for their assigned room
- Moderate messages and users within their room
- Customize room wallpaper and description
- View room-specific analytics

---

## Moderator Permissions

Admins can assign granular permissions to moderators:

- Message deletion
- User muting
- Room management
- User banning
- Each permission can be individually granted or revoked

---

## Maintenance Mode

Admins can enable maintenance mode to temporarily restrict platform access.

- **Immediate maintenance** — Enable right away with a custom message
- **Scheduled maintenance** — Set a future date/time for planned downtime
- **Maintenance banner** — Users see a notification about upcoming or active maintenance

## Best Practices for Moderators

1. **Be fair and consistent** — Apply rules equally to all users.
2. **Warn before acting** — When possible, warn users before muting or banning.
3. **Document actions** — Use admin notes to record why actions were taken.
4. **Escalate when needed** — Contact an admin for situations beyond your permissions.
5. **Stay professional** — Moderators represent the platform; act accordingly.
