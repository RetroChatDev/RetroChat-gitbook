## Troubleshooting

**Airdrop not executing?**
- Check token balance
- Verify recipient list
- Review error messages
- Contact support

**Recipients not receiving?**
- Verify wallet connections
- Check transaction status
- Review recipient list
- Contact support

**Wrong amounts?**
- Verify airdrop details
- Check calculations
- Review recipient list
- Contact support immediately
