### Voting System
- Community members vote
- Upvote/downvote features
- Track vote counts

### Voting Guidelines

- Consider implementation feasibility

### Status Types
- **Under Review**: Being evaluated
- **Planned**: Added to roadmap
- **In Development**: Being built
- **Released**: Available now
- **Declined**: Not planned

### Updates
- Status change notifications
- Development progress
- Release announcements
- Community updates

## Best Practices

1. **Search First**: Check existing requests
2. **Be Specific**: Clear, detailed descriptions
3. **Explain Benefits**: Why it's valuable
4. **Provide Examples**: Help visualize
5. **Engage**: Respond to questions
