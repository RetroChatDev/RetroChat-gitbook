# Room Creation & Management

Create and manage chat rooms on RetroChat.

## Create a Room

### Step-by-Step

1. Navigate to Rooms section
2. Click "Create Room"
3. Enter room details:
   - Room name
   - Description
   - Category
   - Privacy settings
4. Set room rules (optional)
5. Click "Create Room"

### Room Settings

**Basic Settings**
- Room name and description
- Category selection
- Room image/banner
- Tags and keywords

**Privacy Settings**
- Public: Anyone can join
- Private: Invite only
- Unlisted: Hidden from browse

**Moderation Settings**
- Auto-moderation level
- Message filters
- User restrictions
- Content policies

## Invite Users

### Invitation Methods

**Direct Invite**
1. Go to room settings
2. Click "Invite Users"
3. Search for users
4. Send invitations
5. Users receive notification

**Invite Link**
1. Generate invite link
2. Share link with users
3. Users click to join
4. Automatic approval (if enabled)

**Room Code**
1. Generate room code
2. Share code with users
3. Users enter code to join
4. Code expiration settings

### Invitation Settings
- Auto-approve invitations
- Require approval
- Invite expiration
- Maximum members

## Troubleshooting

**Can't create room?**
- Check account status
- Verify permissions
- Try refreshing
- Contact support

**Invitations not working?**
- Verify user exists
- Check privacy settings
- Try different method
- Contact support

**Room settings not saving?**
- Refresh the page
- Check permissions
- Clear cache
- Contact support
