# User Profiles

Customize your RetroChat profile and connect with other users through the follow system.

## Customization

Make your profile unique with extensive customization options.

### Profile Elements

**Avatar**
- Upload a custom profile picture
- Supported formats: JPG, PNG, GIF, WebP
- Recommended size: 200x200 pixels
- Animated GIFs supported

**Banner**
- Add a banner image to your profile
- Recommended size: 1200x300 pixels
- Showcase your personality

**Bio**
- Write a short bio about yourself
- Include links to social media
- Express your interests

**Status**
- Set your current status
- Options: Online, Away, Busy, Offline
- Custom status messages

**Theme**
- Choose light or dark mode
- Custom color schemes (coming soon)
- Profile layout options

### Profile Settings

Access your profile settings to:
- Update personal information
- Manage privacy settings
- Control visibility
- Link social accounts

## Follow System

Build your network by following other users.

### Following Users

1. Visit a user's profile
2. Click the "Follow" button
3. See their updates in your feed
4. Get notified of their activity

### Followers

- See who follows you
- Manage follower requests (if private)
- Block users if needed
- View mutual connections

### Activity Feed

- See posts from users you follow
- View their status updates
- Discover new content
- Engage with their activity

### Privacy Settings

Control who can:
- Follow you
- See your profile
- Send you messages
- View your activity

## Best Practices

1. **Keep It Updated**: Regularly update your profile
2. **Be Authentic**: Represent yourself honestly
3. **Respect Privacy**: Don't share sensitive information
4. **Engage**: Follow interesting users and interact
5. **Stay Active**: Keep your status and activity current

## Privacy

- Control what information is public
- Manage who can contact you
- Set profile visibility
- Block unwanted users

## Troubleshooting

**Can't upload avatar?**
- Check file size and format
- Try a different image
- Clear browser cache

**Follow button not working?**
- Check if user has blocked you
- Verify your account status
- Try refreshing the page

**Can't see profile updates?**
- Check privacy settings
- Verify you're viewing the correct profile
- Clear browser cache
