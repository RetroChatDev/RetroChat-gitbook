# Solana Wallet Setup

Connect your Solana wallet to unlock Web3 features on RetroChat.

## Supported Wallets

RetroChat supports the following Solana wallets:

- **Phantom**: The most popular Solana wallet
- **Solflare**: A feature-rich alternative

## Connect Phantom

### Step 1: Install Phantom

1. Visit [phantom.app](https://phantom.app)
2. Click "Download"
3. Install the browser extension
4. Create a new wallet or import an existing one

### Step 2: Connect to RetroChat

1. Log in to your RetroChat account
2. Go to Settings → Wallet
3. Click "Connect Phantom"
4. Approve the connection in Phantom
5. Your wallet is now connected!

### Step 3: Verify Connection

- Your wallet address will appear in your profile
- You can now send and receive tips
- Airdrops will be sent to this address

## Connect Solflare

### Step 1: Install Solflare

1. Visit [solflare.com](https://solflare.com)
2. Click "Get Started"
3. Install the browser extension
4. Create a new wallet or import an existing one

### Step 2: Connect to RetroChat

1. Log in to your RetroChat account
2. Go to Settings → Wallet
3. Click "Connect Solflare"
4. Approve the connection in Solflare
5. Your wallet is now connected!

## Wallet Management

### Switching Wallets

- Disconnect your current wallet
- Connect a different wallet
- Only one wallet can be connected at a time

### Disconnecting

1. Go to Settings → Wallet
2. Click "Disconnect Wallet"
3. Confirm the disconnection

### Security Tips

- **Never share your seed phrase**: RetroChat will never ask for it
- **Verify transactions**: Always review transactions before approving
- **Keep your wallet secure**: Use strong passwords and enable 2FA
- **Backup your wallet**: Store your seed phrase in a safe place

## Troubleshooting

### Connection Issues

**Wallet not detected?**
- Make sure the extension is installed and enabled
- Refresh the page
- Try disconnecting and reconnecting

**Connection rejected?**
- Check if you approved the connection in your wallet
- Make sure your wallet is unlocked
- Try a different browser

**Wrong network?**
- Ensure you're on Solana mainnet
- Check your wallet network settings
- RetroChat uses Solana mainnet only

### Transaction Issues

**Transaction failed?**
- Check your SOL balance for gas fees
- Verify you have enough tokens
- Check network congestion

**Pending transaction?**
- Wait a few moments for confirmation
- Check your wallet for transaction status
- Refresh the page
