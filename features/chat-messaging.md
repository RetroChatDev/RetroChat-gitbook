# Chat & Messaging

RetroChat offers comprehensive messaging features for both public and private communication.

## Public Rooms

Public rooms are community spaces where multiple users can chat together.

### Joining Rooms
- Browse available public rooms
- Click on a room to join
- See who's currently in the room
- View message history

### Room Features
- Real-time messaging
- File attachments
- Emoticon support
- User mentions with @username
- Message reactions (coming soon)

### Room Management
- Room creators can set rules
- Moderators can manage users
- Pin important messages
- Mute notifications

## Direct Messages

Send private messages to other users.

### Starting a DM
1. Click on a user's profile
2. Select "Send Message"
3. Start your conversation

### DM Features
- One-on-one private conversations
- File sharing
- Message history
- Read receipts
- Typing indicators

## File Attachments

Share files with other users in both public rooms and direct messages.

### Supported File Types
- Images (JPG, PNG, GIF, WebP)
- Documents (PDF, DOC, DOCX)
- Audio files (MP3, WAV)
- Video files (MP4, WebM)

### File Limits
- Maximum file size: 10MB
- Images are automatically optimized
- Files are stored securely

## Emoticons

Express yourself with emoticons throughout the platform.

### Using Emoticons
- Click the emoticon button in the message box
- Select from the emoji picker
- Use keyboard shortcuts for quick access
- Custom emoticons coming soon

## Best Practices

1. **Be Respectful**: Follow community guidelines
2. **Stay On Topic**: Keep conversations relevant to the room
3. **Use Mentions Wisely**: Don't spam @mentions
4. **File Sharing**: Only share appropriate content
5. **Privacy**: Remember that public rooms are visible to everyone

## Troubleshooting

**Messages not sending?**
- Check your internet connection
- Verify you're not muted in the room
- Try refreshing the page

**Can't see messages?**
- Check if you're in the correct room
- Verify your account status
- Contact support if issues persist

**File upload failed?**
- Check file size (must be under 10MB)
- Verify file type is supported
- Try a different file
