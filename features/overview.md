# Features Overview

RetroChat offers a comprehensive set of features designed to create an engaging social experience with Web3 integration.

## Core Features

### Chat & Messaging
- **Public Rooms**: Join community chat rooms
- **Direct Messages**: Private one-on-one conversations
- **File Attachments**: Share files with other users
- **Emoticons**: Express yourself with emoji support

### Arcade Games
- Classic retro games integrated into the platform
- Compete with other users
- Leaderboards and achievements

### Radio Streaming
- Browse curated radio stations
- Create and share playlists
- Discover new music

### User Profiles
- Fully customizable profiles
- Follow system to stay connected
- Status updates and activity feeds

## Web3 Features

- Solana wallet integration
- Tipping system
- Airdrops
- Token integration
